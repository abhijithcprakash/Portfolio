Other projects

  1. Image Caption generator

    The Image Caption Generator is a deep learning-based application designed to automatically generate descriptive captions for images. Combining the power of computer vision and natural language processing (NLP), the project showcases how artificial intelligence can interpret visual data and articulate it in human-readable text. The model achieved significant accuracy in generating relevant and concise captions. For instance, an input image of a dog playing with a ball was captioned as: "A dog running with a ball in its mouth on a grassy field."

  2. Driver drowsines detection

    Driver Drowsiness Detection System is a real-time application designed to enhance road safety by monitoring drivers and detecting signs of drowsiness. This project leverages computer vision and machine learning techniques to analyze visual cues and provide timely alerts, reducing the risk of accidents caused by fatigue. This project enhances road safety by providing an automated solution to monitor driver fatigue, thus reducing the number of accidents caused by drowsy driving. The technology is scalable and can be integrated into existing vehicle systems, contributing to the development of smarter, safer roads.

  3. Minute defect Detection

    In this project, I developed several deep learning models to detect a specific, small-type defect occurring on a particular product in a manufacturing process. The challenge was to accurately identify these tiny defects, which required high precision in both model performance and image quality. To address this, I utilized pretrained ImageNet models as a starting point and fine-tuned them to detect the subtle defects. Additionally, I implemented the YOLO (You Only Look Once) object detection algorithm, which is well-suited for real-time detection tasks, to further enhance the model’s accuracy and speed in identifying defects. The model was then deployed on an edge device, the NVIDIA Jetson Nano, which allowed for real-time processing in the manufacturing environment. I integrated necessary cameras and sensors with the Jetson Nano to capture and process images of the product, ensuring optimal conditions for defect detection. This edge deployment reduced the reliance on cloud processing.

  4. Detection of empty slots on an Auto Coner Machine

    The Auto Coner machine, used in the textile industry, has multiple slots where bobbins with wound yarn are placed. The feeding process is typically done manually by laborers, leading to inefficiencies and potential errors. To address this, we developed an object detection model that automatically identifies empty slots on the machine. By leveraging computer vision and deep learning techniques, the model detects the presence or absence of bobbins in each slot. The system uses a camera to capture real-time images of the machine's slots, which are then processed by the object detection model. The model is trained to detect both occupied and empty slots, providing an accurate count of the empty slots. Based on this information, a new automated system is triggered to feed the required number of bobbins into the empty slots, eliminating the need for manual labor. This automation not only enhances efficiency but also reduces the risk of human error, ensuring consistent production quality.

  5. Arm Exoskeleton for Rehabilitation

    We developed a wearable arm exoskeleton controlled via a mobile app, designed to assist individuals affected by stroke- related paralysis. Many stroke survivors experience arm paralysis, which can be partially treated through physiotherapy. However, the challenges of accessing consistent therapy and the daily difficulties patients face prompted us to create this solution. The exoskeleton allows patients to perform physiotherapy exercises independently at home, guided by the mobile app, reducing the need for continuous in-person therapy sessions. Additionally, it provides assistance in performing daily tasks. The exoskeleton is 3D printed, with movement facilitated by servo motors and gear systems. These motors and sensors are managed by microcontrollers, ensuring precise control and responsiveness for effective rehabilitation and daily task support.
